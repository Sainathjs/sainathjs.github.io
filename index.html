Shardula-ai uses computing and Artificial Intelligence to connect children to their roots, Sanskrit and Sanskriti. We have developed an ecosystem to enable Programming in Sanskrit.

To get started, for the interested, a 5 hrs session spanning 3 to 10 days is introduced. This enables participants to write small programs in Sanskrit, resulting in simple drawings/paintings !!!

We have created an app to write program which requires knowledge of any one Indian script like Devnagri, Kannada, Malayalam etc. One can learn programming here without prior experience in Sanskrit or Programming.

Programs written by 3 children is shared in "WORK" section.
Shardula-ai was started in August 2020 with an aim to to empower society using the Artificial Intelligence as a tool.

Shardula in Sanskrit means Tiger. The tiger cub, our logo, is curious and brave to explore.

We have introduced Sanskrit based programming language. As a measure of simplicity to learn this, a group of children were given 10 classes spanning 30-40 minutes each. And this lead them to paint cute pictures using programming !!!

We have an app with a devnagri keyboard and this enables computing using Sanskrit using Devnagri, or a local script.

Demo 1: Simple Instructions: Data types and data movement.

 3 subjects : Rama, Krishna, Lakshmana.
 2 objects: Mango(Aamraphala) and Chocolate (Chaakaleham).
 2 activities: Arjitavaan (Earned) and Dattavaan (Gave).
Inst 1: Raamaha Chakaleham Arjitavaan.
   This means Raama earned a chocolate.
   Running this instruction creates a Data object Ramaaha of type Raama,
   And an object of type chocolate,
   and assigns the object to Raama.
Inst 2: Krushnaaya Ramaha Chaakaleham Dattavaan
   This means Raama gave chocolate to Raama.
   Running this instruction moves chocolate from Rama to Krushna.
Inst 3: Krushnaha Raamaaya Chaakaleham Dattavaan
   This means Krushna gave chocolate to Raama.
   Running this instruction moves chocolate from Krushna to Raama.
Inst 4: Lakshmanaha Aaamraphalam Arjitavaan
   This means Lakshmana earned a mango.
   Running this instruction creates an object Lakmanaha of type Laksmana.
   and creates an object instance of type Mango.
Inst 5: Lakshmanaha Raamaya Aaamraphalam Dattavaan
   This means Lakshmana gave the mango to Raama
   Running this instruction moves the mango from Laksmana to Raama.

Demo 2 : Mathematical Operations in Sanskrit Programming Language.

 2 subjects : Rama, Krishna.
 1 objects: Chocolate (Chaakaleham).
 2 activities: Arjitavaan (Earned) and Dattavaan (Gave).
Inst 1: Raamaha Thrini Chakalehani Arjitavaan.
   This means Raama earned 3 chocolates.
   Running this instruction creates a Data object Ramaaha of type Raama,
   And an object of type chocolate,
   and assigns the object to Raama.
Inst 2: Raamaha Dwou Chaakalehou Arjitavaan.
   This means Raama earned 2 chocolates.
   This makes Raama own 2 more chocolates. Hence 5 chocolates totally.
Inst 3: Krushnaaya Raamaha Chaakaleham Dattavaan.
   This means Krushna gave (a) chocolate to Raama.
   This makes Krushna own 1 chocolate and Raama 4.

Demo 3: If Else Condition in Sanskrit Programming Language.

 2 subjects : Rama, Krishna.
 1 objects: Chocolate (Chaakaleham).
 2 activities: Arjitavaan (Earned) and Dattavaan (Gave).
Inst 1: Raamaha Thrini Chakalehani Arjitavaan.
   This means Raama earned 3 chocolates.
   Running this instruction creates a Data object Ramaaha of type Raama,
   And three objects of type chocolate,
   and assigns the objects to Raama.
Inst 2: Yadi Raamasya Chaakaleham Asti, Tarhi Raamaha Krushnaaya Chaakaleham Dadaati
   This means if Raama has 3 chocolates.

Demo 4 : If Else condition in True and False case in Sanskrit Programming Language.

 2 subjects : Rama, Krishna.
 1 objects: Chocolate (Chaakaleham).
 2 activities: Arjitavaan (Earned) and Dattavaan (Gave).
Inst 1: Raamaha Thrini Chakalehaani Arjitavaan.
   This means Raama earned 3 chocolates.
   Running this instruction creates a Data object Ramaaha of type Raama,
   And three objects of type chocolate,
   and assigns the objects to Raama.
Inst 2: Yadi Raamasya Thrini Chaakalehaani Santi, Tarhi Raamaha Krushnaaya Chaakaleham Dadaati
   This means if Raama has 3 chocolates.

   This means if Raama has 3 chocolates.

   Running this instruction caused Raama to NOT give chocolate to Krushna
   This is because Raama does NOT have 3 chocolates in the first place

Demo 6: Function call in Sanskrit Programming Language.

 2 subjects : Rama, Krishna.
 2 objects: Chocolate (Chaakaleham), Mango (Aamraphalam).
 2 activities: Arjitavaan (Earned) , Dattavaan (Gave), Katotu (Execute).
Inst 1: Atha Adhyaaya Adhyaaya_1 Prarambha.
   This means a function with name Adhyaaya_1 starts.
   This is similar to start of function definition.
Inst 2: Raamaha thrini chaakaleehani Arjitavaan.
   This means Raama earned 3 chocolates.
Inst 3: Krishnaha Aamraphalam Arjitavaan.
   This means Krishna earned a mango.
Inst 4: Yadi Krishnasya Chaakalehaani Naasti, Tarhi Raamaha Krushnaaya Chaakaleham Dadaati
   This means if Krishna does not have a chocolate.
   then Raama would give one chocolate to Krushna
   Running this instruction caused Raama to give one chocolate to Krushna
   And then Raama would end up having only two chocolates
   And Krishna to have a Mango and a choloclate
Inst 5: Adhyaaya Adhyaaya_1 Samaaptham.
   This means the function with name Adhyaaya_1 finished.
   This is similar to end of function definition.
Inst 6: Adhyaaya_1 Karootu.
   This means call the function with the name Adhyaaya_1.
Inst 7: Adhyaaya_1 Karootu.
   This is a second call to function with the name Adhyaaya_1.
   You would see a different execution flow for Inst4 if Adhyaaya_1 function.

Demo 7: Spatial concept of an object.

 2 subjects : Rama, Lakshmana.
 4 locations: Uttara(North), Agneya(South-East), Dakshina(South), Eeshanya(North-East).
 2 activities: Vasati (resides) , Gacchati (Goes).
Inst 1: Raamaha Uttarey Vasati.
   This means Raama resides in North.
Inst 2: Raamaha Uttaraat Agneyey Gacchati.
   This means Raama goes from North to South-East.
Inst 3: Laksmanaha Dakshinaat Eeshanye Gacchati.
   This means Lakshmana goes from South to North-East.
